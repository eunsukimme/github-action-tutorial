# AWS S3와 CloudFront 연동

---

## AWS CloudFront 는?

- 콘텐츠(HTML, CSS, JS, etc.)를 사용자에게 더 빨리 배포하도록 AWS에서 제공하는 CDN(Content Delivery Network) 서비스
- AWS CloudFront, 즉 CDN을 사용해서 웹 애플리케이션을 배포하면 로드 밸런싱, 속도 및 성능 향상, 보안 등 다양한 이점이 존재
